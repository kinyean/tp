**HireMe is a desktop application for managing your job application details.** While it has a GUI, most of the user interactions happen using a CLI (Command Line Interface).

* If you are interested in using HireMe, head over to the [_Quick Start_ section of the **User Guide**](UserGuide.html).
* If you are interested about developing HireMe, the [**Developer Guide**](DeveloperGuide.html) is a good place to start.
